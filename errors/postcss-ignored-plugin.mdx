---
title: Ignored PostCSS Plugin
---

## Why This Error Occurred

The project's custom PostCSS configuration attempts to configure unnecessary plugins:

- postcss-modules-values
- postcss-modules-scope
- postcss-modules-extract-imports
- postcss-modules-local-by-default
- postcss-modules

## Possible Ways to Fix It

Remove the plugin specified in the error message from your custom PostCSS configuration.

### How do I configure CSS Modules?

CSS Modules are supported in [Next.js' built-in CSS support](/docs/pages/building-your-application/configuring/post-css).
You can [read more about how to use CSS Modules here](/docs/pages/building-your-application/configuring/post-css).
