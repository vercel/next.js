---

[React Server Components](https://react.dev/reference/rsc/server-components) introduce a new way to build web applications. While they improve performance and simplify data fetching, they also change where and how data is accessed, shifting some of the traditional security assumptions for frontend apps.

This guide will help you understand how to approach data security, and how to implement best practices in your Next.js application.
